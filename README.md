# Haskell playground

[![Build Status](https://travis-ci.org/andriy-f/haskell-playground.svg?branch=master)](https://travis-ci.org/andriy-f/haskell-playground)

## Development

1. Set up
```sh
stack setup
```

2. Build
```sh
stack build
```

3. Run app
```sh
stack exec playground-exe
```

4. Run tests
```sh
stack test
```
